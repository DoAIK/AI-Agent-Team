構造的な解決を。

// HOW IT WORKS

4つのフェーズで、抜け漏れを構造的に防ぐ。

マルチフェーズ・チェックアウトが、引き継ぎの全プロセスを構造化します。

PHASE 01 📝

Form Input

AIが業種・職種から最適なチェックリストを自動生成。ハードゲートで必須項目を制御。

PHASE 02 📷

Photo + AI

写真添付とAI画像判定。現場の状態を視覚的に記録し、異常を自動検出。

PHASE 03 🤖

AI Review

Checkout Agentが1項目ずつ対話で確認。危険キーワード検出で自動深掘り。音声対応。

PHASE 04 📋

Document Gen

対話結果から引き継ぎドキュメントとアクションアイテムを自動生成。完全な記録を残す。

// AI AGENTS

3つのAIエージェントが、
引き継ぎを自律的に遂行する。

Google ADK + Gemini を基盤に、各エージェントが専門的な役割を担います。

📑

Template Agent

checklist generator

業種・職種・現場条件からチェックリストを自動生成。過去の引き継ぎログをFew-shot promptingで参照し、最適なテンプレートを提案。

Gemini Firestore ADK
🎤

Checkout Agent

interactive reviewer

1項目ずつAIが対話形式で確認。回答の曖昧さや矛盾を検出し、追加質問を自動生成。Gemini Live APIで音声対話にも対応。

Gemini Live WebSocket NLP
📚

Handover Agent

document generator

チェックアウト結果を解析し、引き継ぎドキュメントとアクションアイテムを自動生成。次のシフトが即座に状況を把握できる。

Gemini PDF Gen ADK

// KEY FEATURES

安全を構造化する機能群。

ヒューマンエラーを仕組みで防ぐ、実践的な機能を搭載しています。

🔒

ハードゲート制御

必須項目を通過しないと次フェーズに進めない。安全確認の形骸化を構造的に防止。

🎲

トラップ質問

「NGが正解」の逆転質問で惰性回答を検出。ランダム順序表示で思考停止を防ぐ。

🎙

リアルタイム音声対話

Gemini Live API (WebSocket) による双方向音声対話。ハンズフリーでチェックアウトを実行。

🚨

危険キーワード自動検出

火災・故障・漏れ等のキーワードを即座に検出。自動で深掘り質問を展開し、リスクを可視化。

📸

写真添付 + AI画像判定

現場写真をAIが分析。目視では見逃しやすい異常兆候を検出し、エビデンスとして記録。

👥

マルチテナント + RBAC

複数拠点・複数チームに対応。ロールベースのアクセス制御で情報セキュリティを確保。
