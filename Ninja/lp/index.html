Problem

引き継ぎは、壊れている。

製造業の交替勤務で繰り返される、構造的な引き継ぎの失敗。ヒューマンエラーではなく、プロセスの問題だ。

抜け漏れ

チェックリストがあっても、項目の確認は自己申告。重要な情報が後任に伝わらず、インシデントが繰り返される。

形骸化

同じチェックリストを毎日繰り返すうち、確認作業がルーティン化。「いつも通り」が思考停止を生む。

暗黙知の喪失

ベテランの経験やノウハウはドキュメントに残らない。人が替わるたびに、組織の知恵が消えていく。

Architecture

3つのAIエージェントが、
引き継ぎを構造化する。

Google ADK上に構築された専門エージェントが、チェックリスト生成から引き継ぎノート作成までを自律的に遂行する。

AGENT 01

Template Agent

チェックリスト自動生成

業種・職種・既存PDFから最適なチェックリストを自動生成。Geminiのマルチモーダル解析で既存ドキュメントをそのまま取り込む。

Gemini 2.0 Flash PDF Parser Firestore
AGENT 02

Checkout Agent

AI対話チェックアウト

チェックリストの各項目をAIが1つずつ確認。危険キーワードを検出すると自動で深掘り質問を展開し、見逃しを防ぐ。

Gemini Live API WebSocket Trap Questions
AGENT 03

Handover Agent

引き継ぎノート自動生成

対話結果を構造化し、引き継ぎノートとアクションアイテムを自動生成。後任が迷わないドキュメントを、AIが書く。

Structured Output Action Items GCS Export

Features

4フェーズ チェックアウト

フォーム入力から自動ドキュメント生成まで、4つのフェーズで引き継ぎ品質を担保する。

構造化フォーム入力

チェックリストの各項目に対して、ステータス選択とコメントを入力。Template Agentが生成した項目に沿って、漏れなく情報を記録する。

自動生成チェックリスト ステータス選択 コメント入力 下書き自動保存

写真AI判定

現場の写真をアップロードすると、Geminiのマルチモーダル解析が自動で状態を判定。目視確認だけでは見逃す異常を、AIが検出する。

Gemini Vision解析 異常検出アラート GCS自動アップロード

AI対話レビュー

Checkout Agentがチェックリストの回答を1項目ずつレビュー。曖昧な回答や危険キーワードを検出すると、自動で深掘り質問を展開する。音声AIにも対応。

逐次レビュー 危険キーワード検出 自動深掘り 音声AI対応
Gemini Live APIによるリアルタイム音声AI対話。テキストと音声のデュアルモードで、現場の状況に合わせた柔軟なチェックアウトを実現。

自動ドキュメント生成

Handover Agentが対話結果を分析し、構造化された引き継ぎノートとアクションアイテムを自動生成。後任は読むだけで状況を把握できる。

引き継ぎノート自動生成 アクションアイテム抽出 優先度自動判定 PDF/HTML出力
Anti-Complacency

形骸化を許さない、5つの防壁。

チェックリストの「慣れ」による思考停止を、多層的な仕組みで防ぐ。

WALL 01

トラップ質問

意図的に誤った前提の質問を挿入。機械的に「はい」と答えると矛盾が検出され、注意を喚起する。

WALL 02

ランダム順序

チェック項目の表示順を毎回ランダムに変更。順番の記憶による惰性確認を防止する。

WALL 03

ハードゲート

クリティカルな項目は「確認済み」だけではPassさせない。具体的な数値や状態の入力を必須とするゲート機構。

WALL 04

AI深掘り

Checkout Agentが回答の曖昧さを検出し、追加質問で掘り下げる。「問題なし」の一言では通過できない。

WALL 05

写真AI判定

テキストだけでなく、写真のエビデンスを求める。Gemini Visionが写真の内容を自動解析し、報告との整合性を検証する。
